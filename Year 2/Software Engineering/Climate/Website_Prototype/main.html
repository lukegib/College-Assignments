Features

Video call - Create a room to 1 on 1 video call friends, family and co-workers via computer, laptop or smart phone.

Text chat - Instant message users in real time without having to worry about an information leak, as your messages don't go anywhere near a server.

File Transfer - File Transfer rooms allow for extremely large files to be transfered peer-to-peer at maximum efficiency and a low latency.

All Peer to Peer - Skype has video call, WhatsApp and Facebook have text chat, DropBox has file transfer, but none of them have complete peer to peer connections meaning data can only be seen by you and your guest.

About Climate

What is Climate?

Climate is an advanced signalling web app powered by real time communications enabling peers to establish browser to browser connections with the ability to share and interconnect data on a decentralised platform.
